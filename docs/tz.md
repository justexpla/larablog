# Че делаем

*запуск (обязательно через CMD терминал):* D:/Programming/Software/ngrok/ngrok.exe http larablog.loc:80


Сайт с функциональностью типа пикабу/твиттера

на начальных этапах:

- Реализуем функционал добавления/редактирования/просмотра посты    [V]
- Создаем сиды  [V]
- Прикручиваем авторизацию и аутентификацию [Для авторизованных и авторов уже есть - надо для модеров]
    (начни с таблиц)
- Просматривание всех постов и постов определенного пользователя

Роли будет 4 штуки
- Админ: может все
- Модератор: создавать посты, редактировать и удалять свои и чужие (но не админские) (soft delete)
- пользователь: создавать посты, удалять и редактировать свои посты
- Забаненный: может редактировать и удалять свои посты, создавать не может
